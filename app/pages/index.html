MyErrata

MyErrata is a collector of web page fixes. You could fix a typo on any web page.

Example

  1. Start editing this page: MyErrata
  2. Modify a text. Save it.
  3. Click on the MyErrata button again. It will stop the editing.

How to fix any web page

MyErrata bookmarklet allows you to activate the editing on any web page. Bookmark or drag the link to your bookmarks bar: MyErrata

See existing fixes.    MyErrata is open source.
